# 重生之我在xxx做xxx - 文字冒险游戏

## 🎮 游戏简介

这是一个手绘漫画风格的重生题材文字冒险游戏，包含多个离谱爽文单元剧：

- **霸道总裁线**：重生为世界首富，掌控全球经济命脉
- **千古一帝线**：重生为开国皇帝，统一天下开创盛世
- **乞丐之王线**：重生为街头乞丐，凭借系统逆袭人生

## ✨ 游戏特色

- 🎯 **属性驱动**：魅力、智力、财富、武力四大属性影响剧情走向
- 🎲 **金手指系统**：每个角色拥有独特的开局能力
- 📖 **第一人称沉浸式**：离谱爽文剧情，越看越上头
- 🎨 **手绘漫画风格**：轻松有趣的视觉体验
- 🔀 **多支线结局**：你的选择决定命运

## 🚀 如何运行

### 方法一：直接打开（推荐）

1. 找到 `index.html` 文件
2. 双击用浏览器打开
3. 开始游戏！

### 方法二：使用本地服务器

```bash
# 进入项目目录
cd rebirth-game

# 使用 Python 启动简单服务器
python3 -m http.server 8080

# 或者使用 Node.js 的 http-server
npx http-server -p 8080
```

然后在浏览器中访问：`http://localhost:8080`

## 🎯 游戏玩法

1. **选择角色**：从霸总、皇帝、乞丐中选择一个重生身份
2. **做出选择**：阅读剧情，点击选项按钮做出决策
3. **属性变化**：你的选择会影响各项属性值
4. **探索支线**：不同的选择会开启不同的剧情线
5. **达成结局**：根据你的选择，解锁不同结局

## 🎨 自定义扩展

### 添加新角色

在 `data.js` 的 `characters` 数组中添加：

```javascript
{
    id: 'new_character',
    name: '新角色名称',
    description: '角色描述',
    initialStats: { charm: 50, intelligence: 50, wealth: 50, strength: 50 },
    cheats: ['金手指1', '金手指2'],
    startNode: 'new_start'
}
```

### 添加新剧情

在 `data.js` 的 `storyNodes` 对象中添加剧情节点：

```javascript
'new_start': {
    text: `剧情文本...`,
    choices: [
        { text: '选项1', nextNode: 'node_1', statCheck: null },
        { text: '选项2', nextNode: 'node_2', statCheck: null }
    ]
}
```

### 修改属性系统

在 `engine.js` 中可以调整属性计算逻辑和范围限制。
